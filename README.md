# X Bookmark(クロスブックマーク)

**X (旧Twitter) のブックマークをもっと便利に。**

X Bookmarkは、Xのブックマーク機能を拡張するChrome拡張機能です。複数アカウントでも一元管理でき、タグ付けや高度な検索で、後から見返すのがとても簡単になります。

## ✨ 主な機能

### 📌 タグ付けで整理

ブックマークにタグを付けて、自分だけの整理方法で管理できます。後から探すときも、タグをクリックするだけで関連するツイートがすぐに見つかります。

### 🔍 高度な検索

ユーザー名、本文、タグ、日付範囲など、様々な条件で検索できます。大量のブックマークの中から、探しているツイートを素早く見つけられます。

### 👥 複数アカウント対応

どのXアカウントでログインしていても、すべてのブックマークを一か所で管理できます。アカウントを切り替える必要はありません。

### 🎨 Xに溶け込むデザイン

Xの標準UIに自然に馴染むデザイン。ダークモード、ライトモードにも対応しています。

### ✍️ ツイート投稿機能

ブックマーク一覧画面から直接ツイートを投稿できます。画像も添付可能です。

## 🚀 使い方

### 1. インストール

#### Chrome / Vivaldi / Microsoft Edge / Opera など

1. https://github.com/yadokari1130/X-Bookmark/releases/latest から`XBookmark_Chrome.zip`をダウンロードします
2. `XBookmark_Chrome.zip`を好きな場所に解凍します
3. [chrome://extensions](chrome://extensions)にアクセスします
4. 開発者モード(デベロッパーモード)が有効になっていない場合は有効にします
5. 「パッケージ化されていない拡張機能を読み込む」を押し、解凍したファイルを選択します

### 2. ログイン

拡張機能のアイコンをクリックして、Googleアカウントでログインします。

### 3. ブックマークを追加

Xのタイムライン上で、各ツイートの下にオレンジ色のブックマークボタンが表示されます。クリックすると、タグを入力してブックマークに追加できます。

### 4. ブックマークを見る

Xのサイドメニューに追加された「X Bookmark」ボタンをクリックすると、保存したブックマークの一覧が表示されます。

### 5. 検索・絞り込み

一覧画面の検索バーで、キーワードやタグで絞り込めます。タグをクリックすると、そのタグが付いたブックマークだけを表示します。

## 💡 便利な使い方

- **タグをクリック**: ブックマーク一覧でタグをクリックすると、そのタグで検索できます
- **画像の拡大表示**: ツイートの画像をクリックすると、大きく表示されます
- **アクションボタン**: 各ブックマークから直接リツイート、いいね、共有ができます
- **並び替え**: 新しい順・古い順で並び替えられます
